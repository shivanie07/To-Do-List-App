## To-Do List App
The To-Do List App is a simple and interactive web application that helps users organize their daily tasks efficiently.
It allows users to add, edit, delete, and mark tasks as complete, while storing all data locally in the browser using Local Storage, ensuring tasks persist even after page refresh.

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Storage: Browser Local Storage
- Tools: Git, VS Code, Browser(Chrome, Microsoft Edge etc.)

## Folder Structure
- index.html
- style.css
- script.js

## Features
1. Add new tasks easily.
2. Edit existing tasks.
3. Mark tasks as complete.
4. Delete single or all tasks at once.
5. Persist data using Local Storage.
6. Fully responsive design for desktop and mobile.
7. View due-date-based task highlights on a dynamic calendar. 
8. Search tasks or filter by completion status.
9. Toggle between light and dark themes with persistent user preference.
